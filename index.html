Automation Modelling and Control Slide Template

This slide deck provides a boiler plate to get started.

  • All of the setup is done. Simply add work between the div class="slides".
  • The menus, page numbering, chalkboard, ...etc. Are automatically included and ready.
  • The following sections provide simple starter cases to start.
  • There are a lot more plugins for this framework. Add them, and send a pull request. I will review and merge
  • HTML5 will provide nearly any interaction you want:
    • Text, images, audio, video.
    • Interactive 3D graphics (e.g. video games).
    • Connection with other instruments.
    • Direct connection with the client video cam and microphone (e.g. Zoom).
